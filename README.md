# 🚀 Startup Intelligence Agent

An AI-powered platform that evaluates a startup's funding readiness by analyzing business information and pitch deck content using Large Language Models (LLMs). The system generates investment insights, identifies strengths and weaknesses, and provides actionable recommendations to help startups improve their chances of securing funding.

---

## 📌 Features

- Analyze startup business details
- AI-powered funding readiness assessment
- Business viability evaluation
- Identify strengths and weaknesses
- Generate investment recommendations
- Interactive web interface
- Automated report generation

---

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Bhagyashree2310/Startup-Intelligence-Agent.git
cd Startup-Intelligence-Agent
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Create a `.env` file

Add your Groq API key:

```
GROQ_API_KEY=your_api_key_here
```

### 4. Run the application

```bash
python main.py
```
