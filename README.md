# 🚀 Startup Intelligence Agent

An AI-powered platform that helps evaluate a startup's funding readiness by analyzing pitch decks and business information using Large Language Models (LLMs). The system generates investment insights, identifies business strengths and weaknesses, and provides actionable recommendations for founders and investors.

---

## 📌 Features

- Upload and analyze startup pitch decks
- AI-powered funding readiness assessment
- Business viability evaluation
- Strengths and weaknesses analysis
- Investment recommendations
- Interactive web interface
- Automated report generation

---

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Bhagyashree2310/Startup-Intelligence-Agent.git
cd Startup-Intelligence-Agent
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Configure environment variables

Create a `.env` file and add your API key:

```
GROQ_API_KEY=your_api_key_here
```

### 4. Run the application

```bash
python main.py
```
